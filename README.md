# Zepto Product Data Analysis (SQL Project)

## Project Overview
This project analyzes product data from Zepto using SQL.  
The dataset contains information about different products available in Zepto including category, product name, price, discount, stock availability and quantity.

The goal of this project is to practice SQL queries and extract useful insights from real-world e-commerce data.

## Dataset Source
The dataset was collected from publicly available product information from kaggle.

## Tools Used
- MySQL
- SQL
- GitHub

## Database Schema

Table Name: zepto

Columns:
- category
- name
- mrp
- discountPercent
- availableQuantity
- discountedSellingPrice
- weightInGms
- outOfStock
- quantity

## SQL Operations Performed

The project includes SQL queries such as:

- Creating database and table
- Importing CSV data
- Data cleaning
- Filtering products
- Finding discounted products
- Stock availability analysis
- Product price analysis
- Aggregation queries

Example Queries:

- Find products with highest discount
- Find products that are out of stock
- Compare MRP vs Discounted price
- Count products by category
